Acciones Recomendadas

  • Auditoría Regular: Revise este dashboard semanalmente para detectar configuraciones incorrectas.
  • Productos Nuevos: Asegúrese de que todos los productos nuevos tengan tax types asignados correctamente.
  • Validaciones: El sistema ahora bloquea la creación de productos sin tax types o con configuraciones incorrectas.
  • Capacitación: Consulte la guía de tipos de impuestos (GUIA_TIPOS_IMPUESTOS.md) para entender cuándo usar cada tipo.
  • Reportes DGII: Antes de generar reportes 606/607, verifique que la puntuación de cumplimiento sea mayor a 95%.
